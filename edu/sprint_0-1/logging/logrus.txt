Logrus

Logrus — это библиотека логирования для Go. Вот некоторые из её преимуществ перед стандартным пакетом log:

    Поддерживает несколько форматов вывода. Вот они: JSON, Text и Syslog.
    Контролирует уровни логирования. В том числе Debug, Info, Warning, Error и Fatal.
    Поддерживает хуки. С их помощью можно настроить, как и куда будут выводиться логи.
    Позволяет настроить формат времени записи логов.

Вот как с помощью Logrus можно записать сообщение Hello, World! в лог уровня Info:

    package main

    import (
        "github.com/sirupsen/logrus"
    )

    func main() {
        log := logrus.New()
        log.Info("Hello, World!")
    }

При запуске этой программы в консоли появится сообщение:

time="2021-10-20T12:00:00+02:00" level=info msg="Hello, World!"
